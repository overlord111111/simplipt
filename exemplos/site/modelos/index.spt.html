Programe em Português

SimpliPT é uma linguagem de programação completa escrita em português. Tipagem opcional, modo debug, servidor web, banco SQLite, template engine. Zero dependências.

▶ Documentação Ver exemplos →
SimpliPT
usar servidor
usar banco

banco.conectar("blog.db")
banco.criar_tabela("CREATE TABLE IF NOT EXISTS posts (id INTEGER PRIMARY KEY, titulo TEXT, conteudo TEXT)")

servidor.rota("GET", "/", fn(req)
    posts = banco.buscar("SELECT * FROM posts")
    retornar servidor.html("index.html", {posts: posts})
fim)

falar servidor.iniciar(3000)

Por que SimpliPT?

Uma linguagem feita pra quem quer produtividade sem perder o controle. Código que parece português, mas tem poder de verdade.

⚖

Sintaxe Natural

Comandos como escrever, se...senao, enquanto, para, tentar...capturar. Código que qualquer um entende.

⚡

Zero Dependências

Não precisa instalar nada além do Python. Cada módulo foi escrito à mão, do zero. Sem node_modules, sem venv gigante.

🔨

18 Módulos Embutidos

Servidor web, banco SQLite, JSON, cache, fila, HTTP, regex, matemática, data, hashes, arquivos, template engine e mais.

🌐

Servidor Web Próprio

Crie APIs REST, sites com templates, sessões, formulários, arquivos estáticos. Tudo com usar servidor.

📜

Template Engine

Arquivos .spt.html com {{ var }}, {% se %}, {% para %}, {% incluir %}. Compila pra Python nativo.

📈

CLI Completa

REPL, build, formatador, transpilador, dashboard, LSP, debug, strict, hot-reload. simplipt --help para tudo.

📜

Tipagem Opcional

Declare tipos com nome: texto = "João". 7 tipos suportados com validação automática em runtime.

🔍

Modo Debug

Depurador passo a passo com breakpoints, inspeção de variáveis, avaliação de expressões. simplipt --debug script.spt.

🔧

Build & Compilar

Compile scripts para executáveis portáteis (.pyz) ou .exe standalone com PyInstaller. simplipt build script.spt.

📋

Modo Estrito

Checagem estática de tipos sem executar o script. simplipt --strict script.spt encontra erros antes do runtime.

🔄

Hot-Reload

Servidor web recarrega rotas automaticamente ao salvar o arquivo. simplipt --hot server.spt para desenvolvimento ágil.

⌨

LSP + VS Code

Servidor LSP para autocomplete e diagnóstico em tempo real. Extensão VS Code com highlight, snippets e comandos integrados.

⚠

Garantir (Assert)

Verificações com garantir(cond, "msg"). Lança erro runtime se a condição for falsa. Ideal para testes e validações.

📂

Scaffolding

simplipt new projeto cria CRUD completo com servidor, banco SQLite, templates e CSS dark, pronto pra usar.

📋

Test Runner

simplipt test [dir] executa todos os .spt como testes, reporta PASS/FAIL/ERRO com tempos.

📈

Profiling

--profile mostra tempo gasto por função e contagem de chamadas. Otimize seus scripts com dados reais.

〈

JSON Mode

--json retorna saída estruturada (sucesso, saida, erros, AST). Integração com outras ferramentas.

🗃

banco.modelo()

ORM embutido: defina o schema e ganhe criar, buscar, atualizar, deletar automaticamente.

🔍

Debugger Web

simplipt debug [dir] — interface gráfica no navegador. Passo a passo, variáveis, pilha, avaliar expressões.

⚡

Async/Await

async funcao fetch() + resp = esperar fetch(). Concorrência nativa com asyncio.

📦

Módulos Externos

simplipt modulo install requests — baixa pacotes Python e usa como módulo SimpliPT.
