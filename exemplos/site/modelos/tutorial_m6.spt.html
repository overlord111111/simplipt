Modulo 6: Web

Crie sites e APIs com o servidor embutido do SimpliPT.

Introdução

Por que web? O SimpliPT tem um servidor HTTP embutido. Zero configuração. O modulo servidor permite criar sites e APIs em poucas linhas, sem precisar instalar Flask, Express ou qualquer outra dependência.

Servidor Web

Para começar, use o modulo servidor e defina rotas com servidor.rota(). Cada rota recebe um metodo HTTP, um caminho e uma função que processa a requisição.

simplipt
usar servidor

servidor.rota("GET", "/", fn(req)
    retornar "Ola, mundo!"
fim)

servidor.iniciar(3000)

Acesse http://localhost:3000 no navegador e veja o resultado. Simples assim.

Rotas e Parametros

Você pode definir rotas para qualquer metodo HTTP: GET, POST, PUT, DELETE. Use :id na URL para capturar parametros dinâmicos.

simplipt
servidor.rota("GET", "/ola/:nome", fn(req)
    retornar "Ola, " + req.params.pegar("nome")
fim)

servidor.rota("POST", "/dados", fn(req)
    retornar "Recebido!"
fim)

Use req.params.pegar("chave") para acessar os valores extraídos da URL.

Templates

Para criar paginas HTML completas, configure uma pasta de templates com servidor.modelos("pasta") e renderize com servidor.html().

simplipt
servidor.modelos("modelos")

servidor.rota("GET", "/pagina", fn(req)
    retornar servidor.html("exemplo.spt.html", {
        titulo: "Minha Pagina",
        itens: ["A", "B", "C"]
    })
fim)

No template, use {{ titulo }} para variaveis e loops com {% para item em itens %} ... {% fim %}. O SimpliPT suporta cabecalho, condicionais e inclusao de arquivos.

APIs JSON

Retornar um dicionario de dados converte automaticamente para JSON. Use req.corpo para acessar dados enviados pelo cliente.

simplipt
mensagens = []

servidor.rota("GET", "/mensagens", fn(req)
    retornar mensagens
fim)

servidor.rota("POST", "/mensagens", fn(req)
    mensagens.adicionar(req.corpo)
    retornar { sucesso: verdadeiro }
fim)

Perfeito para construir APIs REST rápidas.

HTMX Helpers

O template engine do SimpliPT fornece helpers htmx.* para gerar HTML com atributos HTMX — frontend dinâmico sem JavaScript. Eles estão disponíveis dentro dos templates (.spt.html), sem necessidade de importação.

simplipt
<!-- Disponivel em qualquer template .spt.html -->
{{ htmx.botao("Carregar", "/dados", "get", "#alvo") }}
{{ htmx.link("Clique aqui", "/pagina", "get", "#conteudo") }}
{{ htmx.input_texto("nome", "/buscar", "change", "#resultados") }}
{{ htmx.formulario("/enviar", "post") }}
{{ htmx.mostrar("/status", "5") }}
{{ htmx.deletar("/item/1", "#lista", "Tem certeza?") }}

Cada helper gera HTML puro com os atributos hx-*. Basta incluir o script HTMX via CDN no cabecalho. Veja a documentação completa para a lista de todos os helpers e seus parametros.

Exercicios

Abra o Playground e resolva:

1. Ola, Mundo!

Crie uma rota GET / que retorna "Ola, Mundo!".

2. Saudacao Personalizada

Crie uma rota GET /saudar/:nome que retorna "Ola, {nome}!" usando req.params.

3. Pagina com Template

Crie uma rota que renderiza um template HTML com uma lista de itens usando loop {% para %}.

4. Desafio: API de Tarefas

Construa uma API REST de tarefas com GET /tarefas (listar) e POST /tarefas (criar). Use um vetor para armazenar os dados em memoria.
