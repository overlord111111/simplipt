API REST com SQLite

Um CRUD completo de tarefas com servidor web e banco: listar, criar, deletar e atualizar.

api_tarefas.spt
usar servidor
usar json
usar arquivos

ARQUIVO = "tarefas.json"

fn carregar()
    se arquivos.existe(ARQUIVO)
        retornar json.arquivo(ARQUIVO)
    fim
    retornar []
fim

fn salvar(lista)
    json.salvar(ARQUIVO, lista)
fim

# GET /tarefas — lista todas
servidor.rota("GET", "/tarefas", fn(req)
    retornar carregar()
fim)

# POST /tarefas — cria uma
servidor.rota("POST", "/tarefas", fn(req)
    se req.corpo eh nada ou req.corpo.titulo eh nada
        retornar {erro: "JSON invalido"}
    fim
    tarefas = carregar()
    ultimo = 0
    para t em tarefas
        se t.id > ultimo
            ultimo = t.id
        fim
    fim
    nova = {id: ultimo + 1, titulo: req.corpo.titulo, concluido: falso}
    tarefas = tarefas + [nova]
    salvar(tarefas)
    retornar {ok: verdadeiro, tarefa: nova}
fim)

falar servidor.iniciar(3000)

Template String + Loop

Interpolação de texto com {expressao} dentro de strings e iteração sobre listas.

template.spt
usar servidor

fn montar_lista(itens)
    html = "<ul>"
    para i em itens
        html = html + "<li>{i.nome}</li>"
    fim
    html = html + "</ul>"
    retornar html
fim

servidor.rota("GET", "/", fn(req)
    dados = [{nome: "Alice"}, {nome: "Bob"}, {nome: "Carlos"}]
    retornar montar_lista(dados)
fim)

falar servidor.iniciar(3000)

Ternário em Expressão

Condicional inline que funciona em qualquer lugar: atribuição, argumento, lista, dict.

ternario.spt
# Ternário em atribuição
status = "ativo" se idade >= 18 senao "inativo"

# Ternário dentro de chamada de função
falar dobro(10 se 5 > 3 senao 0)

# Ternário dentro de lista
itens = [1 se falso senao 2, 3 se verdadeiro senao 4]
# Resultado: [2, 3]

# Ternário em list comprehension
pares = [n se n % 2 eh 0 senao 0 para n em [1,2,3,4]]
# Resultado: [0, 2, 0, 4]

# Ternário dentro de dicionário
d = {valor: 10 se teste senao 0}

# Ternário aninhado
r = "A" se nota >= 9 senao "B" se nota >= 7 senao "C"

Orientação a Objetos

Classes completas com construtor, métodos, herança e super.

classes.spt
classe Animal
    construtor(nome)
        este.nome = nome
    fim
    metodo falar()
        retornar "..."
    fim
fim

classe Cachorro extende Animal
    construtor(nome, raca)
        super(nome)
        este.raca = raca
    fim
    metodo falar()
        retornar "Au au! Sou {este.nome}"
    fim
fim

rex = novo Cachorro("Rex", "Vira-lata")
falar rex.falar()  # "Au au! Sou Rex"

Modo Debug

Execute scripts passo a passo com breakpoints, inspeção de variáveis e avaliação de expressões.

terminal
# Modo debug interativo
simplipt --debug script.spt

# Com breakpoints nas linhas 10 e 42
simplipt --debug -b 10,42 script.spt

# Comandos do debugger:
#  (enter)   Próxima linha (passo)
#  c         Continuar até próximo breakpoint
#  v         Mostrar variáveis do escopo atual
#  p 2 + 2   Avaliar expressão SimpliPT
#  l         Listar código fonte ao redor
#  q         Encerrar execução

Build + Executável

Compile seu script SimpliPT para um arquivo portátil .pyz ou um executável .exe standalone.

terminal
# Compilar para zipapp (portátil, precisa de Python)
simplipt build app.spt
# Cria: app.pyz
python app.pyz

# Compilar para .exe (standalone, sem Python)
pip install pyinstaller
simplipt build --exe app.spt
# Cria: dist/app.exe
dist/app.exe

Modo Estrito (Strict)

Checagem estática de tipos antes de executar o script. Útil para CI e desenvolvimento.

terminal
# testar.spt
nome: texto = "João"
idade: inteiro = "vinte"  # Erro!

# Verificar sem executar
simplipt --strict testar.spt
# Saída:
#   Tipo incompativel em 'idade': declarado como 'inteiro', valor é 'texto'

Pipe Operator

Encadeie chamadas de funções com |> para código mais limpo.

pipe.spt
fn dobrar(n) retornar n * 2 fim
fn adicionar(n) retornar n + 10 fim
fn fmt(n) retornar "Resultado: {n}" fim

# Sem pipe:
falar fmt(adicionar(dobrar(5)))  # Resultado: 20

# Com pipe (mais legível):
falar 5 |> dobrar |> adicionar |> fmt
# Resultado: "Resultado: 20"
