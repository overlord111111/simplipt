Tutorial / Modulo 1

Fundamentos

Variaveis, tipos de dados, operadores e textos — a base de tudo na programacao.

Introducao

Bem-vindo ao SimpliPT! Se voce nunca programou antes, esta no lugar certo. Programar é dar instrucoes para o computador executar. O SimpliPT foi feito para ser natural: voce escreve em portugues, e o computador entende.

Neste modulo, voce vai aprender os fundamentos que sao a base de qualquer programa: guardar informacoes (variaveis), trabalhar com diferentes tipos de dados (numeros, textos, verdadeiro/falso) e fazer operacoes com eles.

Dica: Nao se preocupe em decorar tudo. Programacao se aprende praticando. Digite os exemplos, experimente, erre e tente de novo.

Variaveis

Variavel é um "nome" que guarda um valor. Pense como uma caixinha com uma etiqueta: voce coloca algo dentro e usa a etiqueta para encontrar depois.

SimpliPT
# Guardando valores em variaveis
nome = "Over"
idade = 30
altura = 1.75
programador = verdadeiro

falar nome
falar idade
falar altura
falar programador

Explicacao linha a linha:

  • nome = "Over" — cria a variavel nome e guarda o texto "Over"
  • idade = 30 — guarda o numero 30
  • altura = 1.75 — guarda um numero decimal
  • programador = verdadeiro — guarda um valor logico
  • falar nome — mostra o valor de nome na tela

O sinal de = é o operador de atribuicao. Ele pega o valor da direita e guarda na variavel da esquerda.

Importante: Nomes de variaveis nao podem comecar com numero e nao podem ter espacos. Use letras, numeros e sublinhado (_). Ex: minha_idade, nomeCompleto.

Tipos de Dados

Toda informacao no computador tem um tipo. Os principais tipos no SimpliPT sao:

SimpliPT
# Tipos basicos
texto   = "Isso eh uma string"     # Texto (tambem chamado de string)
inteiro = 42                       # Numero inteiro
decimal = 3.14                     # Numero decimal
logico  = verdadeiro               # Booleano: verdadeiro ou falso
vazio   = nada                     # Nada (ausencia de valor)

falar texto
falar inteiro
falar decimal
falar logico
falar vazio

Lembrete: Texto sempre entre aspas duplas "assim". Numeros escrevem sem aspas. "42" é texto, 42 é numero.

Operadores

Operadores fazem coisas com os valores. Os principais grupos:

Aritmeticos

SimpliPT
soma    = 10 + 5     # 15
difer   = 10 - 5     # 5
mult    = 10 * 5     # 50
div     = 10 / 5     # 2
resto   = 10 % 3     # 1 (resto da divisao)

falar soma
falar difer
falar mult
falar div
falar resto

Comparacao

Comparam dois valores e retornam verdadeiro ou falso:

SimpliPT
falar 10 > 5       # verdadeiro (10 eh maior que 5)
falar 10 < 5       # falso
falar 10 >= 10     # verdadeiro (10 eh maior ou igual a 10)
falar 10 <= 9      # falso
falar 10 eh 10       # verdadeiro (10 eh igual a 10)
falar 10 nao eh 5    # verdadeiro (10 eh diferente de 5)

Logicos

Combinam condicoes:

SimpliPT
falar verdadeiro e falso         # falso (os dois precisam ser verdadeiros)
falar verdadeiro ou falso        # verdadeiro (um dos dois ja basta)
falar nao verdadeiro              # falso (inverte o valor)

falar (10 > 5) e (5 > 2)         # verdadeiro
falar (10 > 5) e (5 < 2)         # falso
Cuidado: O operador de igualdade é == (dois sinais de igual). Um sinal = é para atribuir, nao para comparar.

Trabalhando com Textos

Textos (strings) sao super versateis. Voce pode juntar, repetir e ate interpolar valores dentro deles.

Concatenacao

Juntar textos com +:

SimpliPT
nome = "Over"
saudacao = "Ola, " + nome + "!"
falar saudacao    # "Ola, Over!"

Template Strings (interpolacao)

Forma mais moderna: use {variavel} dentro do texto:

SimpliPT
nome = "Over"
idade = 30
falar "Meu nome eh {nome} e tenho {idade} anos."
# "Meu nome eh Over e tenho 30 anos."

Voce pode colocar qualquer expressao dentro das chaves:

SimpliPT
a = 10
b = 5
falar "{a} + {b} = {a + b}"
# "10 + 5 = 15"

Ternario (if em uma linha)

O operador ternario valor se condicao senao outro_valor permite escolher entre dois valores em uma linha:

SimpliPT
idade = 18
pode = "sim" se idade >= 18 senao "nao"
falar "Pode dirigir? {pode}"   # "Pode dirigir? sim"

# Funciona em qualquer lugar, inclusive dentro de template strings:
falar "{'maior' se idade >= 18 senao 'menor'} de idade"
Poderoso: O ternario em SimpliPT funciona em qualquer expressao — dentro de strings, argumentos de funcao, listas, etc.
