Projeto 1: Gerenciador de Tarefas CLI

variaveis funcoes arquivos JSON CLI

O que voce vai construir

Um gerenciador de tarefas completo que roda no terminal: adicionar, listar, concluir e deletar tarefas. Tudo salvo em JSON.

O que voce vai aprender

  • Ler e escrever JSON com json.ler() e json.gerar()
  • Criar loops de menu interativo com enquanto
  • Estruturar um programa em funcoes modulares
  • Usar se/senao se para tratar comandos

Estrutura final dos arquivos

texto
tarefas.spt       # codigo principal
tarefas.json      # gerado automaticamente

Passo 1: Estrutura base e persistencia JSON

Crie tarefas.spt. Comece com as importacoes e as funcoes de persistencia:

SimpliPT
usar json
usar arquivos

ARQUIVO = "tarefas.json"

fn carregar()
    se arquivos.existe(ARQUIVO)
        conteudo = arquivos.ler(ARQUIVO)
        retornar json.ler(conteudo)
    fim
    retornar []
fim

fn salvar(lista)
    arquivos.escrever(ARQUIVO, json.gerar(lista, indentado=verdadeiro))
fim

carregar() le o JSON do disco e retorna uma lista vazia se o arquivo nao existir. salvar() recebe a lista e grava formatada no disco.

Passo 2: Listar tarefas

Adicione a funcao que exibe as tarefas com seu status:

SimpliPT
fn listar(lista)
    se tamanho(lista) eh 0
        falar "  Nenhuma tarefa."
        retornar
    fim
    para i ate tamanho(lista)
        t = lista[i-1]
        status = "[X]" se t.feita senao "[ ]"
        falar texto(i) + ". " + status + " " + t.titulo
    fim
fim

Cada tarefa e um dicionario {titulo, feita}. O operador ternario escolhe [X] ou [ ] conforme o status.

Passo 3: Adicionar e concluir tarefas

SimpliPT
fn adicionar(lista)
    titulo = ler("Titulo: ")
    lista.adicionar({titulo: titulo, feita: falso})
    salvar(lista)
    falar "Adicionada!"
fim

fn concluir(lista)
    listar(lista)
    n = inteiro(ler("Numero: "))
    se n >= 1 e n <= tamanho(lista)
        lista[n-1].feita = verdadeiro
        salvar(lista)
        falar "Concluida!"
    senao
        falar "Numero invalido."
    fim
fim

adicionar() pergunta o titulo, insere na lista e salva. concluir() lista, pergunta o numero, marca como feita e salva.

Passo 4: Deletar e menu principal

SimpliPT
fn deletar(lista)
    listar(lista)
    n = inteiro(ler("Numero: "))
    se n >= 1 e n <= tamanho(lista)
        lista.remover(n-1)
        salvar(lista)
        falar "Deletada!"
    senao
        falar "Numero invalido."
    fim
fim

fn main()
    tarefas = carregar()
    enquanto verdadeiro
        falar ""
        falar "=== GERENCIADOR DE TAREFAS ==="
        falar "1. Listar"
        falar "2. Adicionar"
        falar "3. Concluir"
        falar "4. Deletar"
        falar "5. Sair"
        op = ler("Escolha: ")
        se op eh "1"; listar(tarefas)
        senao se op eh "2"; adicionar(tarefas)
        senao se op eh "3"; concluir(tarefas)
        senao se op eh "4"; deletar(tarefas)
        senao se op eh "5"
            falar "Ate mais!"
            retornar
        senao
            falar "Opcao invalida."
        fim
    fim
fim

main()

deletar() usa lista.remover() para remover pelo indice. main() carrega as tarefas, mostra o menu em loop e redireciona para a funcao certa.

Pronto! Como testar

Execute no terminal:

bash
simplipt tarefas.spt

O programa cria tarefas.json automaticamente. Teste: adicione algumas, feche e reabra — os dados persistem.

Dica: Quer ir alem? Adicione uma opcao para editar o titulo, ou prioridades (alta, media, baixa). O JSON aceita qualquer campo extra nos dicionarios.

Passo 1: Servidor basico

Projeto 3: Chatbot de Terminal com Historico

dicionarios regex arquivos loop

O que voce vai construir

Um chatbot interativo que roda no terminal, responde com base em palavras-chave usando regex e salva o historico da conversa em arquivo.

O que voce vai aprender

  • Criar um sistema de respostas baseado em padroes com regex
  • Trabalhar com dicionarios para mapear entradas a respostas
  • Salvar e recuperar historico de conversa em arquivo
  • Implementar um loop de chat com comandos especiais

Passo 1: Dicionario de respostas

Crie chatbot.spt com as respostas e a funcao que encontra a resposta certa:

SimpliPT
usar arquivos
usar regex

respostas = {
    "ola": "Ola! Como posso ajudar?",
    "nome": "Sou o JarbsBot, prazer!",
    "tchau": "Ate logo!",
    "piada": "O que o SQL disse pro NoSQL? Nao me esqueco de voce!"
}

fn responder(texto)
    para chave, valor em respostas
        se regex.testar(chave, texto)
            retornar valor
        fim
    fim
    retornar "Nao entendi. Tente: ola, nome, piada, tchau"
fim

O dicionario mapeia padroes regex a respostas. responder() percorre os padroes e retorna a primeira correspondencia, ou um fallback.

Passo 2: Loop de chat

Adicione o loop principal que interage com o usuario:

SimpliPT
fn main()
    falar "=== JarbsBot v1.0 ==="
    falar "Digite /sair para encerrar"

    enquanto verdadeiro
        entrada = ler("Voce: ")

        se entrada eh "/sair"
            falar "Ate logo!"
            retornar
        fim

        resposta = responder(entrada)
        falar "JarbsBot: " + resposta
    fim
fim

main()

O loop pede entrada, verifica se o usuario quer sair, e responde. Teste com simplipt chatbot.spt.

Passo 3: Comandos uteis

Adicione comandos /historico e /limpar antes do bloco de resposta normal:

SimpliPT
ARQUIVO = "historico.txt"

# dentro do loop, antes de responder:
se entrada eh "/historico"
    falar arquivos.ler(ARQUIVO) se arquivos.existe(ARQUIVO) senao "(vazio)"
senao se entrada eh "/limpar"
    arquivos.escrever(ARQUIVO, "")
    falar "Historico limpo."
senao
    resposta = responder(entrada)
    falar "JarbsBot: " + resposta
fim

/historico exibe o conteudo do arquivo. /limpar o esvazia.

Passo 4: Persistir historico automaticamente

Grave no arquivo cada mensagem da conversa. Adicione no inicio da main():

SimpliPT
    arquivos.adicionar(ARQUIVO, "[INICIO]\n")

E dentro do loop, apos ler a entrada e apos gerar a resposta:

SimpliPT
    # depois de ler a entrada:
    arquivos.adicionar(ARQUIVO, "Voce: " + entrada + "\n")

    # e depois de responder (dentro do senao):
    arquivos.adicionar(ARQUIVO, "JarbsBot: " + resposta + "\n")
