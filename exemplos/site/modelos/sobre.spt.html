Sobre o SimpliPT

Uma linguagem de programação em português, construída do zero.

O que é?

SimpliPT é uma linguagem de programação completa onde você escreve código em português. Não é um tradutor ou uma camada por cima de outra linguagem — é um ecossistema próprio: lexer, parser, AST, interpretador, runtime, CLI e 18 módulos de biblioteca padrão, tudo escrito do zero em Python.

Como funciona

O código SimpliPT passa por quatro estágios:

  • Lexer — Divide o texto em tokens (palavras reservadas como se, para, operadores, números, strings)
  • Parser — Constrói uma Árvore Sintática Abstrata (AST) a partir dos tokens
  • Interpretador — Percorre a AST e executa cada nó, gerenciando escopo de variáveis
  • Runtime — Funções nativas e módulos stdlib disponíveis durante a execução

Stack técnica

  • Python 3.12+ — sem dependências externas
  • 245 testes — cobrindo lexer, parser, interpretador e todos os módulos stdlib
  • ~6000 linhas de código fonte
  • 18 módulos na biblioteca padrão

Filosofia

SimpliPT foi feito para ser prático. Sem firulas, sem camadas de abstração desnecessárias. Código que você lê e entende na hora, em português, e que resolve problemas reais: servidor web, scripts de automação, manipulação de arquivos, banco de dados, APIs.

Quem fez

SimpliPT foi criado por Over, com assistência de Jarbs, um assistente de IA com personalidade forte e acesso total ao sistema. O desenvolvimento ocorreu entre 17 e 18 de junho de 2026.
